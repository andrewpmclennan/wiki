==== Project Structure ====

=== Apps ===
- **ngctv** - Main Android TV application
- **oskservice** - On-Screen Keyboard service
- **dtvlauncher** - DirectTV launcher

=== Module Groups ===
- **common-modules**: account, cloud_dvr, configuration, core, discovery, locationservice, metrics, networkapis, shef, testutil, xCMS
- **directv-now-modules**: common, commonngc, commonplayer, commontv, framework-tv, messaging, ngckeyboard, ngcsettings, quickplayplayer
- **ov-modules**: placeholder
- **testing-modules**: e2e
- **benchmark-modules**: macrobenchmark, microbenchmark

---

== Complete Project Build ==

```bash
# Build everything (all apps and modules)
./gradlew build

# Clean and build everything
./gradlew clean build

# Build without tests (faster)
./gradlew build -x test

# Build with parallel execution
./gradlew build --parallel

# Build with build cache
./gradlew build --build-cache
```

==== App-Specific Commands ====

=== NGCTV App ===

== Build All Variants ==
```bash
./gradlew :apps:ngctv:build
./gradlew :apps:ngctv:assemble
```

== Build Debug Variants ==
```bash
# Atom debug
./gradlew :apps:ngctv:assembleAtomDebug

# Mono debug
./gradlew :apps:ngctv:assembleMonoDebug

# Mono debug with certificate
./gradlew :apps:ngctv:assembleMonoDebugWithCert

# All debug variants
./gradlew :apps:ngctv:assembleDebug
```

== Build Release Variants ==
```bash
# Atom release
./gradlew :apps:ngctv:assembleAtomRelease

# Mono release
./gradlew :apps:ngctv:assembleMonoRelease

# Mono release with certificate
./gradlew :apps:ngctv:assembleMonoReleaseWithCert

# All release variants
./gradlew :apps:ngctv:assembleRelease
```

== Build Benchmark Variants ==
```bash
# Atom benchmark
./gradlew :apps:ngctv:assembleAtomBenchmark

# Mono benchmark
./gradlew :apps:ngctv:assembleMonoBenchmark

# Benchmark release (requires enableBaselineProfile=true)
./gradlew :apps:ngctv:assembleBenchmarkRelease
```

== Build Non-Minified Release (for profiling)
```bash
./gradlew :apps:ngctv:assembleAtomNonMinifiedRelease
./gradlew :apps:ngctv:assembleMonoNonMinifiedRelease
```

== Build All Variants of a Flavor
```bash
# All Atom variants (debug, release, benchmark, etc.)
./gradlew :apps:ngctv:assembleAtom

# All Mono variants
./gradlew :apps:ngctv:assembleMono
```

==== Clean Commands

=== Global Clean
```bash
# Clean entire project (all apps and modules)
./gradlew clean
```

=== App-Specific Clean
```bash
# Clean NGCTV
./gradlew :apps:ngctv:clean

# Clean OSKService
./gradlew :apps:oskservice:clean

# Clean DTVLauncher
./gradlew :apps:dtvlauncher:clean

# Clean all apps
./gradlew :apps:ngctv:clean :apps:oskservice:clean :apps:dtvlauncher:clean
```

=== Module-Specific Clean
```bash
# Clean specific module
./gradlew :modules:common-modules:core:clean
./gradlew :modules:directv-now-modules:commontv:clean

# Clean multiple modules
./gradlew :modules:common-modules:account:clean \
		  :modules:common-modules:cloud_dvr:clean \
		  :modules:common-modules:core:clean
```

```bash
# Clean downloaded Charles certificates (NGCTV specific)
./gradlew :apps:ngctv:cleanCertificates
```

==== Common Workflows

=== Development Workflow
```bash
# Quick incremental debug build (no clean)
./gradlew :apps:ngctv:assembleAtomDebug

# Quick incremental build with parallel execution
./gradlew :apps:ngctv:assembleMonoDebug --parallel

# Build and install on connected device
./gradlew :apps:ngctv:installAtomDebug
./gradlew :apps:ngctv:installMonoDebug
```

=== Clean Build Workflows
```bash
# Clean rebuild of NGCTV debug
./gradlew :apps:ngctv:clean :apps:ngctv:assembleAtomDebug

# Clean rebuild of NGCTV release
./gradlew :apps:ngctv:clean :apps:ngctv:assembleAtomRelease

# Clean rebuild of everything
./gradlew clean build

# Clean rebuild without tests (faster)
./gradlew clean build -x test
```

=== Release Build Workflows
```bash
# Build release with tests
./gradlew :apps:ngctv:clean :apps:ngctv:assembleAtomRelease :apps:ngctv:testAtomReleaseUnitTest

# Build release bundle for Play Store
./gradlew :apps:ngctv:clean :apps:ngctv:bundleAtomRelease

# Build all release variants
./gradlew :apps:ngctv:assembleRelease
```

=== Multi-App Build
```bash
# Build all apps
./gradlew :apps:ngctv:build :apps:oskservice:build :apps:dtvlauncher:build

# Build all apps in parallel
./gradlew :apps:ngctv:build :apps:oskservice:build :apps:dtvlauncher:build --parallel

# Clean and build all apps
./gradlew :apps:ngctv:clean :apps:oskservice:clean :apps:dtvlauncher:clean \
		  :apps:ngctv:build :apps:oskservice:build :apps:dtvlauncher:build
```

./gradlew :apps:ngctv:test :apps:ngctv:connectedAndroidTest

==== Build Variants Reference

=== NGCTV Build Variants

== Product Flavors
- **atom** - Original APK for Osprey platform (optimized for Osprey hardware)
- **mono** - Monolithic APK for AndroidTV/Chromecast/Emulator (APIs 29, 30)
  - Includes ABIs: `armeabi-v7a`, `arm64-v8a`, `x86`, `x86_64`

== Build Types
- **debug** - Debug build with symbols, not obfuscated
- **release** - Production build with ProGuard/R8 minification and resource shrinking
- **benchmark** - Performance benchmarking build (not debuggable)
- **debugWithCert** - Debug build with SSL certificate support (mono only)
- **releaseWithCert** - Release build with SSL certificate support (mono only)
- **benchmarkRelease** - Benchmark release variant (requires `enableBaselineProfile=true`)
- **nonMinifiedRelease** - Release without minification for profiling (requires `enableBaselineProfile=true`)

== Standard Build Variants
- `atomDebug`
- `atomRelease`
- `atomBenchmark`
- `monoDebug`
- `monoRelease`
- `monoBenchmark`
- `monoDebugWithCert`
- `monoReleaseWithCert`

== Conditional Variants (require specific gradle.properties settings)
- `atomBenchmarkRelease` (requires `enableBaselineProfile=true`)
- `atomNonMinifiedRelease` (requires `enableBaselineProfile=true`)
- `monoBenchmarkRelease` (requires `enableBaselineProfile=true`)
- `monoNonMinifiedRelease` (requires `enableBaselineProfile=true`)

=== OSKService Build Variants
- `debug`
- `release`

=== DTVLauncher Build Variants
- `debug`
- `release`

==== Performance Tips

=== Speed Up Builds
```bash
# Use parallel execution (recommended)
./gradlew build --parallel

# Skip tests for faster builds
./gradlew build -x test

# Use Gradle daemon (usually enabled by default)
./gradlew build --daemon

# Use build cache
./gradlew build --build-cache

# Combine for maximum speed (clean build without tests)
./gradlew clean build -x test --parallel --build-cache
```

=== Configuration in gradle.properties
```properties
# Enable parallel builds
org.gradle.parallel=true

# Enable caching
org.gradle.caching=true

# Increase JVM heap size
org.gradle.jvmargs=-Xmx24g

# Enable configuration cache (Gradle 8+)
org.gradle.configuration-cache=true
```

==== Troubleshooting

=== Common Issues

== Build Fails with "Out of Memory"
```bash
# Increase heap size in gradle.properties
org.gradle.jvmargs=-Xmx24g -XX:MaxMetaspaceSize=2048m
```

== Dependencies Not Resolved
```bash
# Refresh dependencies
./gradlew --refresh-dependencies build

# Clear Gradle cache
rm -rf ~/.gradle/caches/
./gradlew build
```

== Incremental Build Issues
```bash
# Do a clean build
./gradlew clean build
```

== Multiple Gradle Daemons
```bash
# Check daemon status
./gradlew --status

# Stop all daemons
./gradlew --stop

# Then rebuild
./gradlew build
```
